Variables

Variables are the Containers for storing the data.Variables can be used to store multiple data. Generally Variables consists of three parts

  1. Variable Declaration

    2. Variable Declaration is used to declare a variable

    Syntax: var a;

  2. Variable Assignment

    3. Variable Assignment is to assign a value to a variable

    Syntax: a=10;

  3. Variable Initialization

    4. Variable Initialization means here both Variable Declaration and Variable Assignment will be done at a time

    Syntax: Var a=15;

    Variables can be declared as x=5 means x stores the value 5

Operators

Operators are those which are used to perform operation on the data. Depending on its type, an operator manipulates an arithmetic or logical value, or operand, in a specific way to generate a specific result.

Types of Operators includes:

  1. Arithemetic Operators
  2. Assignment operators
  3. Logical Operators
  4. Comparision Operators

Arithemetic Operators

Arithemetic Operators are used to perform mathematical operation on the given data. Arithemetic operators include:

  1. Addition +

    2. Used to perform Addition between two variables

  2. Subtraction -

    3. Used to perform Subtraction between two variables

  3. Multiplication *

    4. Used to perform Multiplication between two variables

  4. Modulus %

    5. Used to perform Modulus operation

  5. Division /

    6. Used to perform Division operation

Control Structures consists of:

Loops

Loops are used to execute a block of code more than one time. Loops are of 3 types

For, While, Do-While

Syntax of for loop: for (initialization; testing condition; increment/decrement) {
            statement(s)
        }
        
Syntax of while loop: while (boolean condition) 
            {
            loop statements...
            }
        
Syntax of Do-While Loop: do {
            Statements...
        }
        while (condition);
