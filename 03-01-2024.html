JAVASCRIPT

Javascript Introduction Javascript is a single threaded light weighted scripting language. Javascript is dynamically assigned or run time assigned.

Most of the web pages uses Javascript because it provides functionality to the web pages and provides possibility for dynamic manipulation of browser and document.

Javascript consists of two special features.

  1. DOM
  2. BOM

DOM is used to manipulate html Document

BOM is used to manipulate the Browser.

Javascript is easily understandable language.

Javascript is compatible it can be used in frontend and backend.

Javascript executed line by line and It uses interpreter for execution.

Javascript can be involved in to html in two ways

  1. Internally
  2. Externally

Internally means we can write the script tag directly in the body tag

Externally means script will be written in .js file and is attached to html file in the last line of body tag

Variables Variables are the containers for storing the data. They are used to store and retrieve and to manipulate the data.
  1. Variable Declaration

    2. Variable Declaration is used to declare a variable

    Syntax: var nameofthevariable;

    ex: var a;

    Var is a keyword used to declare the scope the variable.

  2. Variable Assignment

    3. Variable Assignment is to assign a value to a variable. Before assigning a value it is undefined and then we reassign to a value.

    Syntax: variablename=value;

    ex: a=10;

  3. Variable Initialization

    4. Variable Initialization means here both Variable Declaration and Variable Assignment will be done at a time

    Syntax: Var a=15;

    Variables can be declared as x=5 means x stores the value 5

Hoisting

In Javascript all the variable declarations will goes to top of the code even before execution.This process is called as Hoisting. Only the declaration will goes up and declaration in initialization will goes up.

Rules for Declaring a variable:

Variable names are case-sensitive, meaning that x and X are different variables.

Variable names must start with a letter, underscore (_), or dollar sign ($).

Variable names cannot contain spaces.

Variable names cannot start with a number.

A variable name cannot be a reserved keyword.
